Task-level walkthroughs that build on the [Get started](/ecosystem/appkit/get-started) basics. Each page solves one product problem with copy-pasteable code.

## Task index

| Task                                                              | What it covers                                                              |
| ----------------------------------------------------------------- | --------------------------------------------------------------------------- |
| [Connect to a wallet](/ecosystem/appkit/howto/connect-to-a-wallet) | Drop-in button, custom connector picker, vanilla flow, and error handling. |
| [Read balances](/ecosystem/appkit/howto/read-balances)             | Toncoin, jetton, and address-explicit reads from React and core.           |
| [Send Toncoin](/ecosystem/appkit/howto/send-toncoin)               | Transfer Toncoin with a component, a hook, or a core action.               |
| [Send jettons](/ecosystem/appkit/howto/send-jettons)               | Transfer jettons and look up jetton metadata.                              |
| [NFTs](/ecosystem/appkit/howto/nfts)                               | Read, render, and transfer NFTs. Build the request offline.                |
| [Sign data](/ecosystem/appkit/howto/sign-data)                     | Sign text, binary payloads, and TON cells.                                 |
| [Use UI widgets](/ecosystem/appkit/howto/use-ui-widgets)           | Use AppKit components for connect, send, balance, NFT, and progress flows. |

## Conventions

Every task page assumes an `AppKit` instance and the React provider are already mounted. If not, start with [Get started](/ecosystem/appkit/get-started). Code examples use real package imports and the same patterns as the official example app.
