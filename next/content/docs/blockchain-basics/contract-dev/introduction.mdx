To start developing smart contracts on TON, use our Rust toolkit, [Acton](https://ton-blockchain.github.io/acton/docs/welcome).

The recommended smart contract language is [Tolk](https://docs.ton.org/tolk/overview).

In the sections below, you can also learn about the following:

- IDEs and editor plugins: [VS Code](/blockchain-basics/contract-dev/ide/vscode) and [JetBrains IDEs](/blockchain-basics/contract-dev/ide/jetbrains)
- Techniques: [signing messages](/blockchain-basics/contract-dev/techniques/signing), [contract sharding](/blockchain-basics/contract-dev/techniques/contract-sharding), and more
- [Blueprint](/blockchain-basics/contract-dev/blueprint/overview) (for legacy developers)
