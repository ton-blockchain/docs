## What is TON?

TON is a blockchain platform for building scalable _decentralized applications_. They are implemented as _smart contracts_ that communicate by exchanging messages.

TON is designed for high throughput and horizontal scalability through sharding a multi-chain architecture. To learn more, see [Foundations](/foundations/core-concepts).

## TON networks

TON provides two public networks:

- **Mainnet**: The production network that uses real Toncoin
- **Testnet**: A testing environment for development and experimentation

These networks are operated by distributed _nodes_ and _validators_ that maintain consensus on the blockchain state. You can learn more in [Foundations](/foundations/core-concepts).

To read on-chain data, use TON [blockchain explorers](/ecosystem/explorers/overview#explorers-comparison).

## Toncoin

**Toncoin** is TON’s native cryptocurrency used to pay for transactions and smart contract execution.

If you're new to TON, get Toncoin before starting development:

1. Create a wallet. You can use [wallet.ton.org](/ecosystem/wallet-apps/web) or [Tonkeeper](/ecosystem/wallet-apps/tonkeeper).

1. [Get testnet Toncoin](/ecosystem/wallet-apps/get-coins).
