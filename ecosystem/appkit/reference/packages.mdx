Use only public package entrypoints. Do not import from internal source paths.

| Package                          | Exports                                                            |
| -------------------------------- | ------------------------------------------------------------------ |
| `@ton/appkit`                    | Core AppKit, actions, connectors, providers, utilities, and types. |
| `@ton/appkit/queries`            | Query options, mutation options, and query keys.                   |
| `@ton/appkit/swap/omniston`      | Omniston swap provider.                                            |
| `@ton/appkit/swap/dedust`        | DeDust swap provider.                                              |
| `@ton/appkit/staking/tonstakers` | Tonstakers staking provider.                                       |
| `@ton/appkit-react`              | React provider, hooks, components, and `@ton/appkit` re-exports.   |
| `@ton/appkit-react/styles.css`   | Default styles.                                                    |

## Import rules

```ts
import { AppKit, Network, transferTon } from '@ton/appkit';
import { getBalanceQueryOptions } from '@ton/appkit/queries';
import { AppKitProvider, useTransferTon } from '@ton/appkit-react';
import '@ton/appkit-react/styles.css';
```

React package exports core APIs for convenience, but reference pages keep core and React APIs separate.
