`AppKit` is the central runtime object.

## Constructor

### AppKit

Use this constructor to create an AppKit instance.

| Parameter | Type           | Required | Description                                                     |
| --------- | -------------- | -------- | --------------------------------------------------------------- |
| `config`  | `AppKitConfig` | Yes      | Networks, connectors, providers, and defaults for this runtime. |

Returns: `AppKit`.

## Common access patterns

| Pattern      | Use                                                              |
| ------------ | ---------------------------------------------------------------- |
| Core actions | Pass the instance as the first argument.                         |
| React        | Pass the instance to `AppKitProvider`.                           |
| Managers     | Access domain managers such as swap and staking through actions. |
