Core actions are functions exported from `@ton/appkit`. Most actions take an `AppKit` instance as the first argument.

## Action inventory

| Method                  | Purpose                                 |
| ----------------------- | --------------------------------------- |
| `connect`               | Connect through a configured connector. |
| `disconnect`            | Disconnect a wallet session.            |
| `getConnectors`         | Read configured connectors.             |
| `getSelectedWallet`     | Read the selected wallet.               |
| `setSelectedWalletId`   | Set the selected wallet.                |
| `getBalanceByAddress`   | Read Toncoin balance by address.        |
| `watchBalanceByAddress` | Watch Toncoin balance by address.       |
| `getJettonInfo`         | Read jetton metadata.                   |
| `getJettonBalance`      | Read jetton balance.                    |
| `transferJetton`        | Request a jetton transfer.              |
| `getNftsByAddress`      | Read NFTs by owner.                     |
| `transferNft`           | Request an NFT transfer.                |
| `transferTon`           | Request a Toncoin transfer.             |
| `sendTransaction`       | Send a prepared transaction request.    |
| `getTransactionStatus`  | Read transaction status.                |
| `signText`              | Request a text signature.               |
| `signBinary`            | Request a binary signature.             |
| `signCell`              | Request a cell signature.               |
| `getSwapQuote`          | Request a swap quote.                   |
| `buildSwapTransaction`  | Build a swap transaction.               |
| `getStakingQuote`       | Request a staking quote.                |
| `buildStakeTransaction` | Build a staking transaction.            |

## Method format

### transferTon

Use this function to send Toncoin from the selected wallet. On success, a wallet transaction response is returned.

| Parameter    | Type                    | Required | Description                                                 |
| ------------ | ----------------------- | -------- | ----------------------------------------------------------- |
| `appKit`     | `AppKit`                | Yes      | AppKit instance.                                            |
| `parameters` | `TransferTonParameters` | Yes      | Recipient, amount, payload, and optional transfer metadata. |

Returns: `Promise<TransferTonReturnType>`.

### sendTransaction

Use this function to send a prepared transaction request through the selected wallet.

| Parameter    | Type                        | Required | Description          |
| ------------ | --------------------------- | -------- | -------------------- |
| `appKit`     | `AppKit`                    | Yes      | AppKit instance.     |
| `parameters` | `SendTransactionParameters` | Yes      | Transaction request. |

Returns: `Promise<SendTransactionReturnType>`.

### connect

Use this function to connect through a configured connector.

| Parameter    | Type                | Required | Description                               |
| ------------ | ------------------- | -------- | ----------------------------------------- |
| `appKit`     | `AppKit`            | Yes      | AppKit instance.                          |
| `parameters` | `ConnectParameters` | Yes      | Connector and connector-specific options. |

Returns: `Promise<ConnectReturnType>`.
