React components are exported from `@ton/appkit-react`.

## Provider components

### AppKitProvider

This component provides an `AppKit` instance to hooks and components.

| Field      | Type        | Description                              |
| ---------- | ----------- | ---------------------------------------- |
| `appKit`   | `AppKit`    | AppKit instance.                         |
| `children` | `ReactNode` | React subtree that can use AppKit hooks. |

### I18nProvider

This component provides localized strings to AppKit UI components.

| Field      | Type        | Description                  |
| ---------- | ----------- | ---------------------------- |
| `children` | `ReactNode` | React subtree.               |
| `messages` | `object`    | Optional. Message overrides. |

## UI primitives

| Component                     | Use                               |
| ----------------------------- | --------------------------------- |
| `Block`                       | Layout block.                     |
| `Button`                      | Button primitive.                 |
| `CircleIcon`                  | Circular icon wrapper.            |
| `TonIcon`                     | TON icon.                         |
| `TonIconCircle`               | TON icon in a circular container. |
| `BalanceBadge`                | Balance display.                  |
| `CurrencyItem`                | Currency row.                     |
| `NftItem`                     | NFT row.                          |
| `Send`                        | Send flow root.                   |
| `SendTonButton`               | Toncoin send button.              |
| `SendJettonButton`            | Jetton send button.               |
| `TransactionProgress`         | Transaction progress container.   |
| `TransactionProgressProvider` | Progress state provider.          |
