`@ton/appkit/queries` exports query helpers for applications that use TanStack Query or the React bindings.

## Helper types

| Helper           | Use                                                     |
| ---------------- | ------------------------------------------------------- |
| Query options    | Build a query object for read operations.               |
| Mutation options | Build a mutation object for wallet or provider actions. |
| Query keys       | Share cache keys across core and React code.            |

## Cache rules

- Key queries by network, wallet address, asset address, and provider where relevant.
- Invalidate balances after successful transaction requests, then verify on-chain state.
- Do not treat cached provider data as final settlement.
