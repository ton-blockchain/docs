`@ton/appkit-react` exports hooks for AppKit state, actions, queries, and mutations.

## Hook inventory

| Hook                       | Purpose                          |
| -------------------------- | -------------------------------- |
| `useConnect`               | Connect a wallet.                |
| `useDisconnect`            | Disconnect a wallet.             |
| `useSelectedWallet`        | Read selected wallet.            |
| `useConnectedWallets`      | Read connected wallets.          |
| `useBalanceByAddress`      | Read Toncoin balance by address. |
| `useJettonsByAddress`      | Read jettons by address.         |
| `useTransferJetton`        | Request jetton transfer.         |
| `useNftsByAddress`         | Read NFTs by address.            |
| `useTransferNft`           | Request NFT transfer.            |
| `useTransferTon`           | Request Toncoin transfer.        |
| `useSendTransaction`       | Send a prepared transaction.     |
| `useTransactionStatus`     | Read transaction status.         |
| `useSignText`              | Request text signature.          |
| `useSwapQuote`             | Request swap quote.              |
| `useBuildSwapTransaction`  | Build swap transaction.          |
| `useStakingQuote`          | Request staking quote.           |
| `useBuildStakeTransaction` | Build staking transaction.       |

## Hook format

### useTransferTon

Use this hook to request a Toncoin transfer from React.

| Parameter    | Type                       | Required | Description                     |
| ------------ | -------------------------- | -------- | ------------------------------- |
| `parameters` | `UseTransferTonParameters` | Optional | Mutation options and callbacks. |

Returns: `UseTransferTonReturnType`.

### useSelectedWallet

Use this hook to read the selected wallet.

| Parameter    | Type                          | Required | Description    |
| ------------ | ----------------------------- | -------- | -------------- |
| `parameters` | `UseSelectedWalletParameters` | Optional | Query options. |

Returns: `UseSelectedWalletReturnType`.
