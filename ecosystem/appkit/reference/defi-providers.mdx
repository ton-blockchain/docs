DeFi providers connect AppKit to swap and staking services.

## Public provider entrypoints

| Entrypoint                       | Exports                                                                                 |
| -------------------------------- | --------------------------------------------------------------------------------------- |
| `@ton/appkit/swap/omniston`      | `OmnistonSwapProvider`                                                                  |
| `@ton/appkit/swap/dedust`        | `DeDustSwapProvider`, `isDeDustQuoteMetadata`                                           |
| `@ton/appkit/staking/tonstakers` | `TonStakersStakingProvider`, `PoolContract`, `StakingCache`, `createTonstakersProvider` |

## Usage

```ts
import { OmnistonSwapProvider } from '@ton/appkit/swap/omniston';
import { createTonstakersProvider } from '@ton/appkit/staking/tonstakers';

const appKit = new AppKit({
  providers: [new OmnistonSwapProvider(), createTonstakersProvider()],
});
```

## Rule

Quotes are provider data. The transaction and final settlement still need normal wallet approval and verification.
