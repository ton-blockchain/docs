Use this page to request one Toncoin transfer from a connected wallet.

## Before you begin

You need a selected wallet and a recipient address. Use testnet first.

## React flow

```tsx
import { useTransferTon } from '@ton/appkit-react';

export function SendTestTon() {
  const transferTon = useTransferTon();

  return (
    <button
      onClick={() =>
        transferTon.mutate({
          recipientAddress: 'EQ...',
          amount: '0.05',
          comment: 'Test payment',
        })
      }
    >
      Send test TON
    </button>
  );
}
```

## What success means

The wallet accepted the request. It does not prove final settlement. Track the transaction and verify the result before delivering paid goods.

## Common failures

| Failure              | Meaning                                                      |
| -------------------- | ------------------------------------------------------------ |
| User rejected        | The user closed or rejected the wallet request.              |
| Insufficient balance | The wallet cannot pay amount and fees.                       |
| Expired request      | The transaction deadline passed before approval.             |
| Invalid address      | The recipient address is malformed or for the wrong network. |
