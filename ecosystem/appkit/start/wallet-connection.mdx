Use this page to connect a wallet and read the selected wallet state.

## Before you begin

Configure at least one connector. The default browser integration uses TON Connect.

## React flow

```tsx
import { useConnect, useConnectors, useDisconnect, useSelectedWallet } from '@ton/appkit-react';

export function WalletConnection() {
  const { data: connectors = [] } = useConnectors();
  const { data: wallet } = useSelectedWallet();
  const connect = useConnect();
  const disconnect = useDisconnect();

  if (wallet) {
    return (
      <section>
        <p>{wallet.address}</p>
        <button onClick={() => disconnect.mutate()}>Disconnect</button>
      </section>
    );
  }

  return connectors.map((connector) => (
    <button key={connector.id} onClick={() => connect.mutate({ connector })}>
      Connect {connector.name}
    </button>
  ));
}
```

## Common failures

| Failure         | Meaning                                                    | Recovery                                             |
| --------------- | ---------------------------------------------------------- | ---------------------------------------------------- |
| No connector    | AppKit was created without a usable connector.             | Check `connectors` in config.                        |
| User rejected   | The wallet displayed the request and the user rejected it. | Keep the app state unchanged and let the user retry. |
| Session expired | Stored connector state is no longer valid.                 | Disconnect and request a new connection.             |
| Wrong network   | The connected wallet is not suitable for the current flow. | Show the expected network and block unsafe actions.  |
