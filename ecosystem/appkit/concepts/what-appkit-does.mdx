AppKit is the application layer for TON wallet interaction in web apps.

## Responsibilities

AppKit handles:

- wallet connection orchestration;
- selected wallet state;
- TonConnect connector integration;
- transaction and signing requests;
- asset helpers for Toncoin, jettons, and NFTs;
- DeFi provider integration for swaps and staking;
- React hooks and components built on the same core.

## Boundaries

AppKit does not replace:

- wallet security;
- smart contract validation;
- backend order verification;
- RPC provider reliability;
- product-specific business logic.

## Production rule

The browser asks for wallet actions. The backend verifies business outcomes.
