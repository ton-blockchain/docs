AppKit works with Toncoin, jettons, and NFTs.

## Asset model

| Asset   | Meaning                                   |
| ------- | ----------------------------------------- |
| Toncoin | Native TON currency.                      |
| Jetton  | Fungible token standard on TON.           |
| NFT     | Non-fungible token contract and metadata. |

## Metadata rule

Metadata can be delayed, missing, or untrusted. Use it for display. Use contract addresses and verified data for settlement.
