`@ton/appkit-react` wraps core actions and state with React providers, hooks, and components.

## Layers

| Layer            | Description                                                            |
| ---------------- | ---------------------------------------------------------------------- |
| `AppKitProvider` | Provides an AppKit instance to hooks and components.                   |
| Hooks            | Query and mutation wrappers around core AppKit behavior.               |
| Components       | UI primitives and flow components.                                     |
| Styles           | Default component styles imported from `@ton/appkit-react/styles.css`. |

## Rule

Use core actions for framework-independent logic. Use React hooks when state must participate in React rendering.
