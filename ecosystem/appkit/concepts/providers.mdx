Providers connect AppKit to external data or DeFi services.

## Provider types

| Type               | Use                                                      |
| ------------------ | -------------------------------------------------------- |
| API client         | Reads blockchain data such as balances and transactions. |
| Streaming provider | Watches live updates.                                    |
| Swap provider      | Returns swap quotes and builds swap transactions.        |
| Staking provider   | Returns staking data and builds staking transactions.    |

## Rule

Provider responses are external data. Validate critical outcomes before changing product state.
