A network defines chain id, API client configuration, and environment assumptions.

## Mainnet and testnet

Use testnet while developing transaction flows. Use mainnet only when recipients, amounts, verification, and error handling are complete.

## Network mismatch

A mismatch can happen when the app expects one network and the wallet or provider is using another. Block unsafe actions and show the expected network.
