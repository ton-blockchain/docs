Examples are complete flows. They show how pages, state, AppKit actions, and backend verification fit together.

| Example                                                                   | Demonstrates                                                     |
| ------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| [Minimal React app](/ecosystem/appkit/examples/minimal-react-app)         | Provider setup, wallet connection, and first state read.         |
| [Event ticket checkout](/ecosystem/appkit/examples/event-ticket-checkout) | Wallet checkout, Toncoin payment, jetton option, and NFT ticket. |
| [Jetton checkout](/ecosystem/appkit/examples/jetton-checkout)             | Token metadata, decimals, and transfer request.                  |
| [NFT ticket](/ecosystem/appkit/examples/nft-ticket)                       | NFT display and ticket ownership.                                |
| [Swap widget](/ecosystem/appkit/examples/swap-widget)                     | Swap quote and transaction build flow.                           |
| [Staking widget](/ecosystem/appkit/examples/staking-widget)               | Staking provider and quote flow.                                 |

Examples are not the source of truth for API definitions. Use [Reference](/ecosystem/appkit/reference) for exact signatures.
