This object represents a connected wallet.

## Fields

| Field         | Type       | Description                      |
| ------------- | ---------- | -------------------------------- |
| `address`     | `string`   | User-friendly wallet address.    |
| `connectorId` | `string`   | Connector that owns the session. |
| `network`     | `Network`  | Optional. Wallet network.        |
| `features`    | `string[]` | Optional. Wallet capabilities.   |

`ts
type Wallet = {
  // See the fields table above.
};
