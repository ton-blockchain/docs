This object describes a wallet transaction request.

## Fields

| Field        | Type                   | Description                                     |
| ------------ | ---------------------- | ----------------------------------------------- |
| `validUntil` | `number`               | Unix timestamp after which the request expires. |
| `messages`   | `TransactionMessage[]` | Messages the wallet should include.             |
| `network`    | `Network`              | Optional. Expected network.                     |

## Shape

`ts
type TransactionRequest = {
  // See the fields table above.
};
`
