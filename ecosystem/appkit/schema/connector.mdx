This object connects AppKit to a wallet transport.

## Fields

| Field        | Type       | Description          |
| ------------ | ---------- | -------------------- |
| `id`         | `string`   | Stable connector id. |
| `name`       | `string`   | Display name.        |
| `connect`    | `function` | Starts connection.   |
| `disconnect` | `function` | Ends connection.     |

## Shape

`ts
type Connector = {
  // See the fields table above.
};
`
