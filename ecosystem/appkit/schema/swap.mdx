This object group describes swap provider data.

## Fields

| Field         | Type                 | Description                                                |
| ------------- | -------------------- | ---------------------------------------------------------- |
| `providerId`  | `string`             | Provider that produced the quote.                          |
| `fromAsset`   | `string`             | Asset being sold.                                          |
| `toAsset`     | `string`             | Asset being bought.                                        |
| `amount`      | `string`             | Amount in human or base units according to provider rules. |
| `transaction` | `TransactionRequest` | Optional. Transaction built from the quote.                |

type Swap = {
  // See the fields table above.
};
