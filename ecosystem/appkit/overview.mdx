AppKit is a TypeScript toolkit for adding TON wallet connection, payments, assets, DeFi flows, and React UI primitives to web applications.

It prepares requests, tracks application state, and connects your interface to wallets and providers. Wallets still sign transactions and data. Your backend still verifies orders, signatures, and business rules.

## What AppKit is

AppKit is the application layer around TON wallet operations. It gives one place to configure networks, connectors, providers, and React integration.

Use it when your app needs to:

- connect a TON wallet;
- request Toncoin, jetton, NFT, swap, staking, or signing operations;
- read wallet data and asset balances;
- reuse the same core logic from plain TypeScript and React.

## What AppKit includes

| Area      | What it provides                                                           |
| --------- | -------------------------------------------------------------------------- |
| Core      | `AppKit`, network configuration, connectors, managers, actions, and types. |
| React     | Provider, hooks, UI primitives, transaction progress helpers, and styles.  |
| Assets    | Balance, jetton, and NFT helpers.                                          |
| DeFi      | Swap and staking provider integration.                                     |
| Reference | Public package exports, methods, hooks, components, and schema objects.    |

## Packages

| Package                          | Use                                                              |
| -------------------------------- | ---------------------------------------------------------------- |
| `@ton/appkit`                    | Core AppKit instance, actions, connectors, providers, and types. |
| `@ton/appkit/queries`            | Query and mutation helpers for data fetching.                    |
| `@ton/appkit-react`              | React provider, hooks, components, and core re-exports.          |
| `@ton/appkit-react/styles.css`   | Default component styles.                                        |
| `@ton/appkit/swap/omniston`      | Omniston swap provider.                                          |
| `@ton/appkit/swap/dedust`        | DeDust swap provider.                                            |
| `@ton/appkit/staking/tonstakers` | Tonstakers staking provider.                                     |

## Current status

AppKit is an application toolkit. Treat wallet approval, transaction inclusion, indexer data, and backend settlement as separate systems. Production apps should verify critical outcomes outside the browser.
