This example demonstrates provider selection, staking quotes, and staking transaction construction.

## Files

`text
src/appkit.ts
src/App.tsx
src/components/
  WalletPanel.tsx
  ExampleFlow.tsx
`

## Run it

1. Install AppKit packages.
1. Configure the TON Connect manifest URL.
1. Configure testnet provider access.
1. Start the app and connect a wallet.
1. Complete the flow on testnet.

## Important code paths

| File             | Role                              |
| ---------------- | --------------------------------- |
| src/appkit.ts    | Creates the AppKit instance.      |
| src/App.tsx      | Mounts providers.                 |
| WalletPanel.tsx  | Connects and disconnects wallets. |
| ExampleFlow\.tsx | Runs the example-specific flow.   |
