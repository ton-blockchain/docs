Use this guide to read NFTs owned by an address and request NFT transfers when supported.

## Before you begin

You need the owner address and a provider that can read NFT data.

## Basic flow

1. Read NFTs by owner address.
1. Display verified fields first.
1. Treat off-chain metadata as display data.
1. Request transfer only when the wallet and item support it.

## Code

```tsx
import { useNftsByAddress, useTransferNft } from '@ton/appkit-react';

export function NftList({ owner }: { owner: string }) {
  const nfts = useNftsByAddress({ ownerAddress: owner });
  const transfer = useTransferNft();

  return nfts.data?.map((nft) => (
    <button key={nft.address} onClick={() => transfer.mutate({ nftAddress: nft.address, recipientAddress: 'EQ...' })}>
      Transfer {nft.name ?? nft.address}
    </button>
  ));
}
```
