Use this guide to request a Toncoin transfer from the selected wallet.

## Before you begin

You need a connected wallet, a recipient address, and a human-readable amount.

## Basic flow

1. Validate the recipient address.
1. Validate the amount and expected network.
1. Request the transfer.
1. Track settlement before delivering value.

## Code

```tsx
import { useTransferTon } from '@ton/appkit-react';

export function PayButton() {
  const transferTon = useTransferTon();

  return (
    <button
      onClick={() =>
        transferTon.mutate({
          recipientAddress: 'EQ...',
          amount: '1.25',
          comment: 'Order #1001',
        })
      }
    >
      Pay 1.25 TON
    </button>
  );
}
```

## Handling errors

| Error                | Recovery                                     |
| -------------------- | -------------------------------------------- |
| User rejected        | Keep order unpaid.                           |
| Insufficient balance | Ask the user to top up.                      |
| Invalid address      | Block the request before opening the wallet. |
