Use this guide to connect a user wallet, read the selected address, and restore the connection after reload.

## Before you begin

You need an AppKit instance, a configured network, and at least one connector.

## Basic flow

1. Read available connectors.
1. Ask the user to choose a connector.
1. Connect and store selected wallet state.
1. Handle disconnect and expired sessions.

## Code

```tsx
import { useConnect, useConnectors, useDisconnect, useSelectedWallet } from '@ton/appkit-react';

export function ConnectWallets() {
  const { data: connectors = [] } = useConnectors();
  const { data: wallet } = useSelectedWallet();
  const connect = useConnect();
  const disconnect = useDisconnect();

  if (wallet) {
    return <button onClick={() => disconnect.mutate()}>Disconnect {wallet.address}</button>;
  }

  return connectors.map((connector) => (
    <button key={connector.id} onClick={() => connect.mutate({ connector })}>
      Connect {connector.name}
    </button>
  ));
}
```

## Handling errors

| Error             | Recovery                                 |
| ----------------- | ---------------------------------------- |
| User rejected     | Keep state unchanged and allow retry.    |
| Missing connector | Check AppKit config.                     |
| Expired session   | Disconnect and request a new connection. |
