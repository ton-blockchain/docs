Use this guide to request text, binary, or cell signatures and verify them outside the browser.

## Before you begin

Define the domain, nonce, expiration time, and payload before opening the wallet.

## Basic flow

1. Build a payload that includes domain and nonce.
1. Ask the wallet to sign.
1. Send the signature to the backend.
1. Verify address, payload, nonce, and expiration.

## Code

```tsx
import { useSignText } from '@ton/appkit-react';

export function SignInButton() {
  const signText = useSignText();

  return <button onClick={() => signText.mutate({ text: 'Sign in to example.com' })}>Sign in</button>;
}
```
