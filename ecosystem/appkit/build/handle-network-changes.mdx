Use this guide to keep UI safe when the app, wallet, or provider network changes.

## Before you begin

Decide which network each product flow supports.

## Basic flow

1. Read configured networks.
1. Read the current network and selected wallet.
1. Disable unsafe actions when the network is wrong.
1. Show a clear recovery path.

## Code

```tsx
import { useDefaultNetwork, useNetwork, useNetworks } from '@ton/appkit-react';

export function NetworkBanner() {
  const networks = useNetworks();
  const current = useNetwork();
  const fallback = useDefaultNetwork();

  if (!current.data) return <p>Select a network</p>;
  return <p>Network: {current.data.name ?? fallback.data?.name ?? networks.data?.[0]?.name}</p>;
}
```
