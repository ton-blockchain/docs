Use this guide to combine wallet connection, price display, transaction request, and backend verification.

## Before you begin

Create orders on the backend. The browser should not decide whether an order is paid.

## Basic flow

1. Create a pending order on the backend.
1. Connect the user's wallet.
1. Display amount, recipient, asset, and expiration.
1. Request the wallet transaction.
1. Verify settlement on the backend.
1. Deliver the ticket or product.

## Code

```tsx
import { useSelectedWallet, useTransferTon } from '@ton/appkit-react';

export function CheckoutButton({ orderId, recipient }: { orderId: string; recipient: string }) {
  const { data: wallet } = useSelectedWallet();
  const transferTon = useTransferTon();

  return (
    <button
      disabled={!wallet}
      onClick={() => transferTon.mutate({ recipientAddress: recipient, amount: '2.5', comment: orderId })}
    >
      Pay order
    </button>
  );
}
```
