Use this guide to show staking providers, quote stake operations, and build staking transactions.

## Before you begin

Register a staking provider such as Tonstakers.

## Basic flow

1. Read staking providers.
1. Request a quote.
1. Show provider, amount, and expected position.
1. Build and send the transaction.

## Code

```ts
import { buildStakeTransaction, getStakingProviders, getStakingQuote } from '@ton/appkit';

const providers = await getStakingProviders(appKit);
const quote = await getStakingQuote(appKit, { providerId: providers[0].id, amount: '10' });
const transaction = await buildStakeTransaction(appKit, { quote });
```
