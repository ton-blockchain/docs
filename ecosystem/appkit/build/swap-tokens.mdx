Use this guide to request swap quotes and build transactions with a configured swap provider.

## Before you begin

Register a swap provider such as Omniston or DeDust.

## Basic flow

1. Ask the provider for a quote.
1. Show amount, route, and expected result.
1. Build the transaction from the selected quote.
1. Send it through the wallet and track settlement.

## Code

```ts
import { buildSwapTransaction, getSwapQuote } from '@ton/appkit';

const quote = await getSwapQuote(appKit, {
  fromAsset: 'TON',
  toAsset: 'JETTON_MASTER_ADDRESS',
  amount: '2',
});

const transaction = await buildSwapTransaction(appKit, { quote });
```
