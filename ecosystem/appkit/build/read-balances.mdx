Use this guide to read Toncoin and asset balances for a wallet address.

## Before you begin

You need a wallet address and a configured API client.

## Basic flow

1. Read native Toncoin balance.
1. Read jetton balances when the UI needs token data.
1. Refresh after transactions.
1. Treat provider data as delayed until confirmed.

## Code

```tsx
import { useBalanceByAddress, useJettonsByAddress } from '@ton/appkit-react';

export function Balances({ address }: { address: string }) {
  const ton = useBalanceByAddress({ address });
  const jettons = useJettonsByAddress({ address });

  return (
    <section>
      <p>TON: {ton.data?.value}</p>
      <p>Jettons: {jettons.data?.length ?? 0}</p>
    </section>
  );
}
```
