Use this guide to track wallet requests and confirm transaction status before delivering value.

## Before you begin

Store the order id, expected recipient, expected amount, user wallet, network, and transaction metadata.

## Basic flow

1. Send the wallet request.
1. Record the pending order.
1. Poll or watch transaction status.
1. Verify the final chain state on the backend.

## Code

```ts
import { getTransactionStatus, watchTransactionsByAddress } from '@ton/appkit';

const status = await getTransactionStatus(appKit, { transactionHash: '...' });

const unwatch = watchTransactionsByAddress(appKit, {
  address: 'EQ...',
  onTransaction: (transaction) => console.log(transaction),
});
```
