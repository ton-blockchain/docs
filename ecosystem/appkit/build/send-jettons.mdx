Use this guide to show jetton balance and request a jetton transfer.

## Before you begin

You need the jetton master address, recipient address, amount, and selected wallet.

## Basic flow

1. Read jetton info and decimals.
1. Read the user's jetton balance.
1. Convert input to the correct amount units.
1. Request the jetton transfer.

## Code

```tsx
import { useJettonBalanceByAddress, useTransferJetton } from '@ton/appkit-react';

export function JettonPayment({ jettonMaster, owner }: { jettonMaster: string; owner: string }) {
  const balance = useJettonBalanceByAddress({ ownerAddress: owner, jettonMasterAddress: jettonMaster });
  const transfer = useTransferJetton();

  return (
    <button
      disabled={!balance.data}
      onClick={() =>
        transfer.mutate({
          jettonMasterAddress: jettonMaster,
          recipientAddress: 'EQ...',
          amount: '10',
        })
      }
    >
      Pay jettons
    </button>
  );
}
```

## Handling errors

| Error                 | Recovery                                              |
| --------------------- | ----------------------------------------------------- |
| Missing metadata      | Show contract address and retry metadata later.       |
| Wrong decimals        | Convert from metadata before building the request.    |
| Missing jetton wallet | Explain that the user has no balance for this jetton. |
