TON Connect is the standard wallet connection protocol for the TON blockchain. It links a dApp to a user's wallet over an end-to-end encrypted session so the app can read the connected address, request signatures, and send transactions — without ever touching the user's keys.

## SDK reference

The dApp-facing packages live in the [`ton-connect/sdk`](https://github.com/ton-connect/sdk) monorepo:

{/* TODO(sdk-ref): switch to custom api ref once merged. */}

- [`@tonconnect/ui-react`](https://ton-connect.github.io/sdk/modules/_tonconnect_ui-react.html) — hooks and prebuilt components for React.
- [`@tonconnect/ui`](https://ton-connect.github.io/sdk/modules/_tonconnect_ui.html) — framework-agnostic UI, same components without React bindings.
- [`@tonconnect/sdk`](https://ton-connect.github.io/sdk/modules/_tonconnect_sdk.html) — headless connector for server-side flows or custom UI.

A working end-to-end example: [demo-dapp-with-react-ui](https://github.com/ton-connect/demo-dapp-with-react-ui) ([live](https://tonconnect-sdk-demo-dapp.vercel.app/)).
