<Aside type="note">
  The official smart contract language of TON Blockchain is [Tolk](/tolk/overview). FunC is now a **legacy** language, with its compiler no longer maintained.

  Learn how to [migrate from FunC to Tolk](/tolk/from-func/tolk-vs-func). For new smart contract projects, use the [Acton toolchain](/contract-dev/acton).
</Aside>

A FunC program is a list of:

- [Function declarations and definitions](/languages/func/functions)
- [Global variable declarations](/languages/func/global-variables)
- [Compiler directives](/languages/func/compiler-directives)
